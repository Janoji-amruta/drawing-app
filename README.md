# 🎨 Drawing App

A simple and interactive drawing application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Draw freely on the canvas using your mouse
- Choose different drawing colors
- Clear the canvas with one click
- Simple and user-friendly interface
- Built using HTML5 Canvas

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- HTML Canvas API

## 📸 Preview

Users can:
- Select a color from the color picker
- Draw on the canvas
- Clear the canvas whenever needed

## 📂 Project Structure

drawing-app/

## ▶️ How to Run

1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Start drawing!
